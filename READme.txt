
ABOUT THE APP:
This aplication uses Unity3D graphics. Coded in C# and JavaScript, using PHP to get
information from the web service. The protocol used to the data transfer was SOAP and 
WSDL. 

To install the app, just download and run the .apk file.

To use the app, tap the button of the planet or mission you want to see closer. Tap
the planet or the mission to know about the coordinates and facts of the mission.
